Improving Lesion Detection in 3D MRI images for Multiple Sclerosis X INRIA

Introduction

This research focuses on improving lesion detection and segmentation in MRI scans for multiple sclerosis patients using deep learning approaches. The goals were to:

  • Enhance the performance of an existing UNet-based model
  • Explore if STIR images could improve feature extraction and model performance
  • Implement ensemble methods and fine-tuning to boost accuracy

Dataset and Preprocessing

The dataset comprised:

  • 255 T2 MRI scans
  • 214 STIR MRI scans
  • Corresponding lesion segmentations

Preprocessing steps included:

  1. Cropping images around the spinal cord
  2. Finding the center of the spinal cord
  3. Shifting the spinal cord to a vertical position

Base Model

The base model used a UNet architecture trained on T2 scans. The best performing base model was trained on 240 T2 images and achieved:

  • Mean lesion sensitivity: 0.743318
  • Mean lesion precision: 0.697821
  • Mean dice coefficient: 0.495983

Ensemble Method

An ensemble approach was implemented using the probability maps (P-maps) from the base model as input. Various machine learning algorithms were tested, with Neural Networks performing best:

  • Mean voxel sensitivity: 0.5682
  • Mean voxel precision: 0.753
  • Mean F1 score: 0.5828

The ensemble method showed improvement over the base model, indicating that STIR images provided valuable additional information.

Fine-tuning

The base model was fine-tuned to incorporate STIR images:

  1. Modified the first convolutional layer to accept both T2 and STIR inputs
  2. Fine-tuned the new layer for 5000 steps
  3. Fine-tuned the entire model for another 5000 steps

Results showed improved sensitivity but lower precision compared to the base model:

  • Mean lesion sensitivity: 0.807045
  • Mean lesion precision: 0.656763
  • Mean dice coefficient: 0.513623

Conclusion

The ensemble method showed the most promise for improving lesion detection and segmentation. Fine-tuning with STIR images improved sensitivity but at the cost of precision. Further investigation into learning rates and training parameters could potentially yield better results.
